# Análises do Projeto da disciplina de MO810: Desigualdade em Campinas – Uma Leitura Interseccional da RAIS

Este repositório reúne os códigos utilizados no projeto **"Desigualdade em Campinas: uma leitura interseccional da RAIS"**, desenvolvido no contexto da disciplina **MO810: Tópicos em Inteligência Artificial – Princípios do Feminismo de Dados**.

Aqui estão armazenados os scripts responsáveis por coleta dos dados, construção de tabelas, gráficos e análises utilizadas no relatório final.

O repositório existe exclusivamente para fins de transparência e reprodutibilidade das análises realizadas.  
Nenhum dado sensível é incluído neste espaço; apenas códigos e artefatos derivados.
